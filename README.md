# 🔐 Secure Registration System (MERN Stack)

A full-stack secure authentication system built with **React, Node.js, Express, and MongoDB**.  
The app allows users to register, login, and access protected routes using **JWT-based authentication**.

---

## 🚀 Features
- User Registration & Login
- JWT Authentication
- Protected Routes (Frontend & Backend)
- Password Hashing (bcrypt)
- REST API (Express)
- Responsive UI (Tailwind CSS)

---

## 🧠 Tech Stack

**Frontend:** React, React Router, Tailwind CSS, Axios  
**Backend:** Node.js, Express.js  
**Database:** MongoDB + Mongoose  
**Authentication:** JWT (JSON Web Tokens)

---

## ⚙️ How It Works
1. User registers an account  
2. Password is hashed and stored securely  
3. User logs in and receives a JWT token  
4. Token is used to access protected routes  
5. Backend verifies token for security
